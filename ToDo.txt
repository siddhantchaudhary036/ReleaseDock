   Beamer, Headway, and FeatureBase all let you pick a future date/time and the

   Canny and FeatureBase let you pull a published post back to draft instantly.
   Add an "Unpublish" button on the edit page for published entries that sets

7. CATEGORIES (SEPARATE FROM LABELS)
   LaunchNotes distinguishes categories (product areas: "iOS", "API", "Billing")
   from labels (change types: "Feature", "Bugfix"). Categories drive subscriber
   preferences and widget filtering. Labels stay as visual tags. Add a new
   categories table, a management UI in settings, and a category picker in the
   editor alongside the existing label chips.

8. WIDGET CATEGORY FILTERING
   LaunchNotes lets you configure each widget embed to show only specific
   categories, so different pages of your app show relevant updates only.
   Add a data-categories attribute to the script tag that filters the feed.
   Dashboard shows a per-widget category selector in widget settings.

9. SEARCH & FILTER ON PUBLIC PAGE
   Canny and FeatureBase public pages have a search bar and category/label
   filter dropdowns. Add a sticky filter bar at the top of the public changelog
   page with a text search input and clickable category/label pills.

10. PAGINATION / INFINITE SCROLL
    Every competitor paginates their feeds. Currently you load all changelogs at
    once. Add cursor-based pagination to the widget API and public page, loading
    ~10 entries at a time with a "Load more" button or infinite scroll trigger.

11. RSS / ATOM / JSON FEED
    Headway, Canny, and AnnounceKit auto-generate RSS feeds for each project.
    Add a /api/feed/:slug route to the HTTP router that returns an RSS 2.0 XML
    document. Link it in the public page header with <link rel="alternate">.
